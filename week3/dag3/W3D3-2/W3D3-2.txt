Opdracht W3D3-2: code review

Als Junior.
Ik vond de junior rol wel prettig. Ik kon uitleggen wat ik had gedaan en waarom. Ik kreeg goede tips om de leesbaarheid van mijn code voor anderen te verbeteren (comments maken). Arno bleef immer vriendelijk, dus ik kan niet zeggen hoe het voelt als iemand op een vervelende manier feedback geeft, maar ik kan me wel voorstellen dat dat niet erg motiverend zal zijn.

Als Senior.
Ik vond deze rol lastiger. Maar ik denk dat ik wel opbouwende feedback heb gegeven. Ik heb ook aangegeven wat ik juist erg goed vond. Net als Arno (en de andere overigens) is het onmogelijk om, zelfs als oefening, een beetje vervelend te doen. Ik vind feedback geven ook lastiger dan feedback ontvangen. Ik vind het lastig om mensen op onvolkomendheden te wijzen. Gelukkig is het makkelijker als het als tip kan worden ingebracht. Ik vindt het zelf ook altijd fijn om een tip te krijgen.

Als observant
Het is leuk om, als een soort vlieg op de muur, te kijken wat de feedback geven/ontvangen stijlen van anderen zijn. Regelmatig denk je dan: 'O, dat is een goede vraag' of 'een goede tip is dat'. Erg leerzaam.
